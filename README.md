# Kringle

Let's make some Christmas donations!

## Installation

Make sure you have [npm](https://www.npmjs.org/) installed, then:

```
$ npm install
$ bower install
```

## Usage

Start the server:
```
$ npm start
```
Then go to ```localhost:5000``` in your browser

## Testing

Tests are written in Mocha.  To run all tests:
```
$ npm test
```
